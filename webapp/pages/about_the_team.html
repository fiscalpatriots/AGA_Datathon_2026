Our Mission

"Create a supplemental resource promoting literacy in audit terminology and data to make government engagement accessible for the general public."

How We Built This

Step 1

Data Integration

We connected three major datasets (USAspending, FAC, and SAM) using the Unique Entity Identifier (UEI) to reveal gaps between funding and exclusion status.

Step 2

Visualization

Using Tableau, we created exploratory dashboards to highlight patterns in funding concentration (Top 10 States) and audit finding signals.

Step 3

Machine Learning

We trained a HistGradientBoosting classifier to predict next-year audit findings based on award complexity and prior history (ROC-AUC: 0.77).

Step 4

Web Development

We built a responsive, accessible web platform using HTML/CSS and deployed it on AWS to make these insights public.
